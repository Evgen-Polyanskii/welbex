## Локальная установка

```
# Клонировать проект
git clone git@github.com:Evgen-Polyanskii/welbex.git

# Перейти в директорию проекта
cd welbex

#Скопировать файл с переменными
cp .envExample .env

# Установка
make setup

```

## Запуск

```
make start
```
Открыть в браузере страницу http://localhost:5000/
